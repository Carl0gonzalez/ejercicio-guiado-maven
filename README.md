# ☕ ejercicio-guiado-maven

Ejercicio guiado en **Java con Maven** pensado para practicar estructura estándar, compilación y gestión de dependencias.

![Java](https://img.shields.io/badge/Java-blue?logo=openjdk) ![Maven](https://img.shields.io/badge/Maven-3.x-orange?logo=apachemaven)

## Descripción

Este repositorio sirve como base de aprendizaje para comprender el ciclo `clean`, `compile`, `test` y `package` en proyectos Maven.

## Objetivo

Reforzar el flujo de trabajo Java profesional a través de una estructura simple y fácilmente reutilizable.

## Uso

```bash
mvn clean test
```
