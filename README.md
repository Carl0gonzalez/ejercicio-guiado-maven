# ☕ ejercicio-guiado-maven

Proyecto guiado en **Java con Maven**, orientado a practicar compilación, estructura estándar y gestión de dependencias.

![Java](https://img.shields.io/badge/Java-blue?logo=openjdk) ![Maven](https://img.shields.io/badge/Maven-3.x-orange?logo=apachemaven)

---

## 📌 Objetivo

Servir como ejercicio base para comprender la organización de un proyecto Java bajo Maven y sus comandos principales.

## ▶️ Ejecutar

```bash
git clone git@github.com:Carl0gonzalez/ejercicio-guiado-maven.git
cd ejercicio-guiado-maven
mvn clean test
# o
mvn clean package
```

## 💡 Valor del repo

Aporta una base clara para practicar estructura Java profesional y automatización de build.
